# MEJORO MI PENSIÓN - Landing Page

Sitio web corporativo de la firma legal **Mejoro Mi Pensión**, especializada en derecho pensional y de seguridad social en Colombia.

## 🚀 Despliegue en Vercel

Este es un sitio web estático (HTML/CSS). Para desplegarlo en Vercel manualmente:

1. Instala la CLI de Vercel: `npm i -g vercel`
2. Ejecuta el comando `vercel` en la raíz del proyecto.
3. Sigue las instrucciones en pantalla.

O simplemente arrastra la carpeta del proyecto al panel de Vercel en el navegador.

## 📁 Estructura del Proyecto

- `index.html`: Página principal.
- `privacidad.html`: Política de privacidad (Ley 1581 de 2012).
- `condiciones.html`: Términos y condiciones del servicio.
- `css/`: Estilos CSS personalizados (basados en el sistema de diseño Premium Legal Excellence).
- `img/`: Activos visuales y logotipos.

## 🛠️ Tecnologías

- HTML5 Semántico
- CSS3 (Vanilla)
- Google Fonts (Playfair Display & Inter)
- Lucide Icons (Inline SVGs)
